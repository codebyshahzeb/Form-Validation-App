# Form Validation Project
A simple and interactive form validation project built using HTML, CSS, and JavaScript.  
This project ensures that users provide valid input before submitting a form, improving the overall user experience and data integrity.

---
## Features

- Real-time Validation: Validates fields as the user types.
- Email Validation: Checks for a proper email format.
- Password Validation: Ensures strong passwords (if implemented).
- Error Messages: Clear and user-friendly error messages.
- Responsive Design: Works smoothly on both desktop and mobile devices.
- Easy to Customize: Add more fields or validation rules easily.

---

## Technologies Used

- HTML5 – Structure of the form  
- CSS3 – Styling with modern design  
- JavaScript (Vanilla JS) – Form validation logic
